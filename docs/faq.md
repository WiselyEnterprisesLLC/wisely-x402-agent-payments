# FAQ

## Is this a public copy of the private Wisely backend?

No. This repository is the public integration package: docs, skill instructions, schemas, examples, and client helpers. The private production server remains private.

## Can any agent use this?

Any agent that can use remote MCP, run a CLI, or make HTTPS calls can use the public surface. OpenClaw, Hermes-style agents, Codex/CLI, and generic MCP clients should all have a path.

## Does Wisely need my seed phrase?

No. Wallet signing stays with the caller's wallet or approved runtime.

## Are Solana, XRPL, and Stellar the same as Base x402?

Base USDC is the canonical native x402 exact settlement rail. Solana, XRPL, and Stellar are Wisely-native adapters with on-chain verification and x402-compatible receipts for Wisely-aware agents.

## How does Wisely make money?

Wisely earns through routed paid usage, hosted endpoint services, creator/tool onboarding, and optional managed support. The public install package is free.
